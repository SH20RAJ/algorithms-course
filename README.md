## 🚀 Features

- **Organized Curriculum**: 14+ chapters covering Greedy, Divide & Conquer, Dynamic Programming, and more.
- **Progress Tracking**: Automatic saving of your progress via `localStorage`.
- **Interactive Player**: Seamless YouTube integration with automatic completion detection.
- **Completion Rewards**: Celebratory confetti and certificates (checkbox based) for every milestone.
- **Premium UI**: Dark mode, glassmorphism, and responsive design for a professional learning experience.

## 📂 Topics Covered

- Introduction to Algorithms
- Analysis of Algorithms
- Divide and Conquer
- Greedy Method
- Dynamic Programming
- Backtracking

- Graph Theory

- And much more...

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript
- **Icons**: FontAwesome
- **Animations**: Canvas Confetti
- **Deployment**: Cloudflare Pages

## 💻 Local Setup

1. **Clone the repo**:
   ```bash
   git clone https://github.com/SH20RAJ/algorithms-course.git
   cd algorithms-course
   ```

2. **Run a local server**:
   You can use `npx serve` or any static file server.
   ```bash
   npx serve .
   ```

3. **Open in browser**:
   Visit `http://localhost:3000`.
